# 🛡️ Automated Fact-Checking Web App

An AI-powered "Truth Layer" web application built for the CogCulture Assessment. This tool automates claim verification by extracting factual statements from uploaded PDF documents and cross-referencing them against live web data using Google Gemini and DuckDuckGo Search.

## 💻 How to Run Locally
1. Install Dependencies: `pip install -r requirements.txt`
2. Run App: `streamlit run app.py`
